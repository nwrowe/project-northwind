# Project Northwind

A mobile-first historical-inspired trade-and-sailing strategy game built in Godot.

## Current Goal
Build a vertical-slice prototype featuring:
- port-to-port trading
- ship supplies and durability
- route risk and travel events
- upgrades
- save/load

## Stack
- Godot 4
- GDScript
- JSON data-driven content

## Project Structure
- `autoload/` global managers
- `scenes/` Godot scenes
- `scripts/` gameplay and UI scripts
- `data/` JSON content definitions

## Next Milestone
Boot into a playable port screen, buy cargo, travel to a new port, resolve an event, and save/load progress.
